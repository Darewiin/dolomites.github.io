About This Project

The Dolomites Adventure Planner is a client-side travel planning dashboard that dynamically calculates trip costs, compares travel months, and generates personalized itineraries for groups visiting the Italian Dolomites. Built as a portfolio project demonstrating full-stack frontend skills.

Technical Features

  • Pure HTML, CSS, and JavaScript — no frameworks
  • Dynamic cost calculation engine
  • Canvas-based donut chart rendering
  • Responsive glassmorphism design
  • Interactive form with real-time validation
  • Deployable directly to GitHub Pages
